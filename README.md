# AbideTogetherSupport
Abide Together is an iOS app designed to help Catholic families and couples stay organized and spiritually grounded. It brings together essential daily tools into one simple, beautiful place â€” so families can focus on what matters most.

Features: Unified Calendar -Add family events and work days -Save commonly used names -Choose colors for each person -Add shift start/end times -Add optional notes -Fully editable entries Notes -Keep and organize important family notes like grocery lists Verse of the day -scripture displayed on the home screen for daily reflection Recipes -Save and share recipes with your family -Organize family meals easily Prayer Library -Collection of prayers for different moments of the day -Accessible anytime Family & Friends Integration -Optionally connect with family members -Shared data only appears for users you add

Tech Stack -SwiftUI -Firebase Authentication -Firebase Firestore -Firebase Storage -MVVM Architecture

Privacy -All user data is private and securely stored in Firebase. -Data is shared only with family or friends explicitly added by the user.
